### Chord / bar notation

- **Nashville-style scale degrees** — harmony is `1`–`7` with `b` / `#` where needed; **not** fixed letter names in the grid. Interpret degrees against the **key** given at the top of each chart.
- **Bars** — `[` starts a bar (4/4 unless the chart says otherwise). Symbols inside a bar are separated by spaces.
- **Quality** — suffixes after the degree: e.g. `m`, `D` (dominant 7), `M` (maj7), `ø`, extensions (`9`, `11`, …), `add9` / `add11`, etc.
- **Slash chords** — bass as `/` + scale degree (e.g. `1/3`, `b7/1`).
- **Duration** — one beat per chord by default; `.` extends the current chord (`..` = +2 beats, `...` = +3). The chord lasts until a new degree or a rest.
- **`%`** — repeat the chord pattern from the **previous bar** (with the durations that follow).
- **Rests** — `-` (and `--`, `-.`) mark rests / rhythmic holes.
- **`(text)`** on a chord line — performance note for that bar.
